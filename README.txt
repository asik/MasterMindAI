This programs plays a version of Mastermind called "Hells" which is found the in the video game "Neverwinter Nights 2 : Mask of the Betrayer". It uses a greedy algorithm to find a solution in a small (but not necessarily optimal) number of moves.

I used the program to help me through my Let's Play of the game, which can be found here: http://www.youtube.com/watch?v=vA3XPlTSYoo
